Image2PDF
Converts a bunch of images to a single pdf file.
This is a android-only release.
In order to use, add image files using the search button.
That will show a preview of the image.
Then use the Add (+) button to add the image to the list Adds an image to the list.
Finally, click the ok (tickmark) button to create a pdf
